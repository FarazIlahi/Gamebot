# Discord Game Bot

A Discord bot built using Python that adds interactive and fun features to a server.

## 🚀 Features

* Responds to user events (join/leave)
* Custom commands (e.g. purge messages)
* Interactive chat functionality (planned LLM integration)
* Easy to extend with new commands

## 🛠️ Tech Stack

* Python
* discord.py
* VS Code

## 📂 Project Structure

```
Discord-Gamebot/
│── bot.py
│── commands/
│── utils/
```

## ⚙️ Setup

1. Clone the repository:

```
git clone https://github.com/FarazIlahi/Discord-Gamebot.git
```

2. Install dependencies:

```
pip install discord.py
```

3. Run the bot:

```
python bot.py
```

## 🔐 Environment Variables

Make sure to add your Discord bot token securely:

```
TOKEN=your_discord_token_here
```
